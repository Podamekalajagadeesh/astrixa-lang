# ASTRIXA Language (astrixa-lang)

ASTRIXA Language is the core programming language
of the ASTRIXA ecosystem.

## Why ASTRIXA Exists
Modern developers are forced to use multiple languages
and complex tools to build Web, Web3, and AI systems.

ASTRIXA exists to unify these domains under one
simple, safe, and high-performance language.

## Who ASTRIXA Is For
- Web developers
- Web3 & blockchain engineers
- AI & ML engineers
- Startup founders
- Solo builders

## Core Goals
- Simple and readable syntax
- Strong safety guarantees
- High performance by default
- Native support for Web, Web3, and AI
- Predictable tooling

## What ASTRIXA Refuses To Be
- Not framework-dependent
- Not overly abstract
- Not hard to learn
- Not controlled by a single company

## Long-Term Vision
ASTRIXA aims to become a universal development
language powering applications, protocols,
and intelligent systems worldwide.
